# Minas Dev Events

Repositório central de eventos da comunidade Minas Dev

## Como Adicionar um Evento?

Para adicionar um evento e contribuir com este respositório, [crie uma issue](https://github.com/minasdev/minasdev-events/issues) utilizando o seguinte template:

#### Titúlo da Issue:
[Nome do Evento] / [Data do Evento]

#### Conteúdo da Issue:
```
**Evento: [Nome do Evento](URL)**
**Local: [Local](URL do Google Maps)**
**Data: YYYY-MM-DD**
Rápida descrição
```
Exemplo:

#### Titúlo da Issue:
4º FEMUG-MG / 2017-01-26

#### Conteúdo da Issue:
**Evento: [4º FEMUG-MG](https://goo.gl/YtgxGs)**
**Local: [CI&T Belo Horizonte](https://goo.gl/maps/f9LynEyr2Pr)**
**Data: 2017-01-26**
O FEMUG trata-se de um evento para pessoas interessadas em aprender, evoluir e dividir conhecimentos relacionados a front-end e tecnologias periféricas.
